=== Unterstützung für AWS GovCloud (US) in NetApp Workload Factory

NetApp Workload Factory unterstützt AWS GovCloud (US)-Konten. Mit einem GovCloud-Konto können Sie GenAI-Workloads in den AWS GovCloud (US)-Regionen bereitstellen und verwalten.
